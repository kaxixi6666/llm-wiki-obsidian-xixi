# prompt 与上下文如何塑造 AI-generated SATD？

## 1. Question
- Core question: prompting context、task scope 与 project context 如何影响 AI-generated SATD 的表达和修复？
- Short version: SATD 更像 agent 差异，还是更像上下文效应？
- Why it matters: 作者明确写出：SATD resolution effectiveness appears driven more by prompting context and task scope than by the agent itself.
- Why it is hard: 我的解释：当前来源能支持方向性判断，但无法给出细粒度因果拆解。

## 5. Current State of Understanding
- What seems supported: 作者明确写出：prompting context 和 task scope 对 SATD 修复效果很重要。
- What seems unsupported: 哪一种 prompt / context 因素起主导作用，论文没有明确验证。
- What is still unclear: training data、agent reasoning strategy、prompt formulation 各自的影响如何拆分。
- What may be overclaimed in the literature: 我的解释：如果只按 agent 名称归因，可能忽略了 task scope 和 context 的更强作用。

## 6. My Current Hypothesis
- Main hypothesis: 我的解释：AI-generated SATD 的表达与修复更受 prompt、context、task scope 影响，而不是单纯由 agent 品牌决定。
- Alternative hypothesis: 我的解释：不同 agents 的内在能力差异仍是主要因素。
- Confidence level: medium
- Why I currently think so: 我的解释：当前论文在结果与 future work 中多次强调 prompt/context/task scope。
