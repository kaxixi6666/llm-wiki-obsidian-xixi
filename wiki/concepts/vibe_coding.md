# Vibe Coding

## Definition in Current Sources
- 作者明确写出：在 vibe coding 中，开发者通过自然语言表达高层意图，并更多通过结果观察而不是逐行代码理解来验证 AI 生成实现。
- 作者明确写出：该范式与 autonomous coding agents、prompt-based interaction、prototype-driven workflows 密切相关。
