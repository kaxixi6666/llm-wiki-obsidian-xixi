# Self-Admitted Technical Debt

## Definition
- 作者明确写出：SATD comments 会显式揭示 acknowledged limitations 和 deferred fixes，并影响长期维护。
- 我的解释：在这篇论文里，SATD 被当作研究 AI coding agents 如何表达技术债的主要观察对象。

## In This Paper
- 作者明确写出：论文比较了 AI-authored SATD 与 human-authored SATD 的 prevalence 和 quality。
- 作者明确写出：论文分析了 AI-authored SATD 的 fixing behavior 和 topic types。
